# Teste Técnico – Estágio IntuitiveCare 2026

Este projeto foi desenvolvido como parte do teste de nivelamento do processo seletivo
para Estágio na IntuitiveCare.

O objetivo é coletar dados públicos da ANS, realizar tratamento básico,
organização das informações e análise simples dos dados.

Devido ao prazo reduzido, a solução prioriza clareza, simplicidade e
organização do código.

## Etapa atual
No momento, foi realizada a análise inicial dos arquivos disponibilizados pela ANS,
com o objetivo de compreender a estrutura dos dados antes da implementação
do tratamento automatizado.

## Leitura inicial dos dados

Foi criado um script em Python para leitura inicial dos arquivos CSV da ANS,
permitindo a visualização das colunas e compreensão da estrutura dos dados
antes do processamento completo.

## Filtragem de despesas

Foi implementado um script para filtrar despesas relacionadas a eventos e sinistros
com base na descrição das contas contábeis, gerando um arquivo consolidado
para posterior análise.
