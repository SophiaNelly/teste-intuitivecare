# Teste Técnico – Estágio IntuitiveCare 2026

Este projeto foi desenvolvido como parte do teste de nivelamento do processo seletivo
para Estágio na IntuitiveCare.

O objetivo é coletar dados públicos da ANS, realizar tratamento básico,
organização das informações e análise simples dos dados.

Devido ao prazo reduzido, a solução prioriza clareza, simplicidade e
organização do código.
